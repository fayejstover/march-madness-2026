Algorithm Reference — v2
// Round-adaptive P_base — weights shift each round P_base = weighted_avg( RoundSurvival[seed][round], // 38% → 7% (R64 → Champ) EfficiencyRating / 100, // 20% → 38% RecentForm / 10, // 15% → 26% RegionalDifficulty / 10, // 10% → 5% CoachingFactor / 10 // 17% → 24% ) P_base *= ProgramModifier // Elite programs: +2–6% P_base *= PathFactor // Close wins: -2%; blowouts: +1.5% P_base *= ClusterBoost // Upset clustering: +4–7% P_base *= SeasonTrend // 2026 upset-heavy: weaker seeds +5% WinProb(A vs B) = P_base(A) / (P_base(A) + P_base(B))

v2 improvements over v1:

  • Round survival rates replace flat seed history — #2 seeds drop to near-zero championship probability matching 20-yr historical data
  • Upset timing modifiers — 12-5 upsets boosted in R64/R32 and penalized post-Sweet 16; 6-11 and 7-10 upsets peak in R32
  • Program modifiers — Duke (+6%), Kansas/UConn/Gonzaga (+5%), and 8 other elite programs baked in
  • Path dependency — close-game penalty (−2% per win), blowout boost (+1.5%), cumulative fatigue after 3–4 games
  • Upset clustering — each upset in a region raises the next upset's probability by 4–7%
